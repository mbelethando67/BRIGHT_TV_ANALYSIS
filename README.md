📊 BrightTV Viewership Analysis
📌 Overview

This project analyzes BrightTV’s user viewership data to uncover usage trends, identify factors influencing content consumption, and recommend strategies to increase engagement and grow the subscriber base.

🎯 Business Objective

BrightTV aims to grow its subscription base by improving user engagement and optimizing content strategy.

🧰 Tools Used
SQL (BigQuery / Databricks)
Microsoft Excel
Google Looker Studio (Dashboard)
🗂️ Dataset
User Profiles
Viewership Session Data (10,000 records)
⚙️ Data Processing
Converted timestamps from UTC to South African time
Cleaned and joined datasets
Achieved 100% match rate between user and session data
Aggregated data by:
Channel
Time
Demographics
📈 Key Insights
Sports content dominates viewership, accounting for 31% of total sessions
Peak viewing time occurs during evening hours (18:00–21:00)
Male users contribute 88% of total engagement
Gauteng leads with 36.5% of total users
Lower engagement observed during weekday daytime hours
📉 Opportunities Identified
Underutilized engagement during low-traffic periods
Low penetration in smaller provinces
Opportunity to grow female audience through entertainment content
🚀 Recommendations
Promote sports highlights and replays during off-peak hours
Introduce short-form and on-demand content
Expand marketing in underserved provinces
Personalize content recommendations
Launch referral and promotional campaigns
